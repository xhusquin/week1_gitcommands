Commandes Git et Github

Sommaire

  1. Installation de Git sur Linux avec un paquet apt
  2. Configurer Git
  3. Initialiser un dossier de travail

Installer Git (Debian)

Vérifier le statut 'installed' de Git:
apt list git
Si besoin, installer le paquet:
sudo apt install git

Configurer Git

Vérifier la configuration de Git:
git config --list
Si besoin:
git --global
