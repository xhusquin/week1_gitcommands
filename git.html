Installer Git (Debian)

Vérifier le statut de Git (installed):

apt list git

Si besoin, installer le paquet:

sudo apt install git

Configurer Git

Vérifier la configuration de Git:

git config --list

Indispensable:

git config --global user.name nom
git config --global user.email ton_mail

Optionnel:

git config --global color.diff auto
git config --global color.branch auto
git config --global color.status auto

avec Bash, j'ai dû suivre cette procédure et l'adapter pour faire apparaître le nom de ma branche git dans mon terminal.

Initialiser un dossier de travail

Après avoir créer et naviguer vers un dossier de travail avec les commandes cd et mkdir :

git init

Cloner un repertoire (depuis GitHub)

Copier le lien du dépôt sur GitHub
Naviguer vers le répertoire désiré puis: 

git clone [coller ici le lien]

Il est possible de rajouter un nouveau nom de dossier après le lien copié.

Indexer les fichiers (stage)

git status

pour voir les fichiers à indexer.

git add nom_du_fichier

pour indexer.

Valider la version (Commit)

git commit -m "message de commit"

Mettre à jour le dépôt distant

git push
