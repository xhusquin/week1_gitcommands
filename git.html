Installer Git (Debian)

Vérifier le statut installed de Git:
apt list git
Si besoin, installer le paquet:
sudo apt install git

Configurer Git

Vérifier la configuration de Git:
git config --list
Indispensable:
git config --global user.name nom
git config --global user.email tonmail
Optionnel:
git config --global color.diff auto
git config --global color.branch auto
git config --global color.status auto

avec Bash, j'ai du suivre cette procedure et l'adapter pour faire apparaitre le nom de ma branche git dans mon terminal.

Initialiser un dossier de travail

Après avoir créer et naviguer vers un dossier de travail avec les commandes cd et mkdir :
git init

Cloner un repertoire (depuis Github)

Copier le lien du dépot sur Github
Naviguer vers le repertoire désiré puis:
git clone [coller ici le lien]
Il est possible de rajouter aprés le lien un nouveau nom pour le dépot.

Indexer les fichiers (stage)

git status pour voir les fichiers à indexer.
git add nomduficher pour indexer.
On peut utiliser git add * pour indexer tous les fichiers.
